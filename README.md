# Имитационное моделирование системы массового обслуживания

## Описание проекта

В работе рассматривается система массового обслуживания типа **M/M/2/4**, моделирующая работу автомойки:

- входной поток — пуассоновский,
- время обслуживания — экспоненциальное,
- число каналов обслуживания — 2,
- максимальное число заявок в системе — 4 (включая очередь).

## Цель работы

- Построить аналитическую модель СМО.
- Рассчитать основные характеристики системы.
- Реализовать имитационную модель на Python.
- Сравнить аналитические и экспериментальные результаты.
- Ввести метрику оценки совпадения теории и моделирования.

## Реализация

Аналитическое решение получено для модели M/M/c/K.  
Имитационная модель реализована методом следующего события (next-event simulation).
